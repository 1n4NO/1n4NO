**Featured**

### ⚙️ [Fluxion](https://github.com/1n4NO/fluxion)
A BPMN workflow engine and HTTP service written in Rust. Deploy process definitions, run persistent instances, and orchestrate human and automated tasks — all via a clean REST API.

- Multi-tenant PostgreSQL persistence with migration-on-startup
- Kafka integration for process triggers and lifecycle event publishing
- JWT / OAuth2 (RS256) authentication with role-based access control
- Prometheus metrics + OpenTelemetry distributed tracing (OTLP/gRPC)
- Web console with process graph visualization and instance inspection
