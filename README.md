### Hi, I'm Pratik 👋

I build AI systems, developer tools, and accessibility-focused products — mostly in TypeScript.

Current focus: multi-agent architectures, local LLM orchestration, and making the web more usable.

---

**What I've been building**

- 🧠 [multi-agent-ai-system](https://github.com/1n4NO/multi-agent-ai-system) — production-grade AI pipeline with Planner → Researcher → Writer → Critic agents, real-time SSE streaming, and a React Flow execution graph. Runs fully offline on Ollama.
- ♿ [colorblind-filters-extension](https://github.com/1n4NO/colorblind-filters-extension) — Chrome extension to simulate color vision deficiencies for accessibility testing
- 🔍 [ux-audit-tool](https://github.com/1n4NO/ux-audit-tool) — automated UX & accessibility audits via heuristic rules, built with Next.js and Cheerio
- 🌬️ [rainmatter-air-extension](https://github.com/1n4NO/rainmatter-air-extension) — live AQI readings in your toolbar via OAQ and OpenAQ v3
